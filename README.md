# LANCOM

基于PyQt5+Flask开发的局域网通信与文件传输，使用websocket技术实现，支持同时多人聊天，并且
支持传输的内容如下所示：

1、文字传输

2、图片传输

3、视频传输（new）

****

**将来需要实现的功能**

1、自定义右键（长按）菜单栏

****

**优点**
> 1、一机多用（在window启服务后，就可以通过浏览器进行跨多平台通信）。
> 
> 2、支持方便局域网数据传输，传输敏感数据，防止敏感数据泄露至互联网。

### 安装

```python
pip install -r requirements.txt
```

### 食用方式

直接运行`main.py`文件

```python
python main.py
```

**注：**

- 如果怕安全问题的话，记得改服务器会话密钥
- 在关闭服务的时候，记得刷新状态多关闭一次
